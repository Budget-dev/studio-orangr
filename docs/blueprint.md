# **App Name**: Shyama Overseas

## Core Features:

- Multi-page Navigation: Robust navigation system allowing users to access different sections and pages of the website, including dropdown menus for service categories.
- Dynamic Service & Portfolio Display: Showcase of trade services and export portfolio projects, each with dedicated cards including icons, titles, descriptions, and call-to-actions.
- Client Testimonials & Insights: Displays client feedback in a structured, scrollable format and a blog section for sharing expert insights, news, and export best practices.
- Animated Statistics Counters: Interactive, animated display of key company statistics (e.g., clients served, countries reached) that activate upon visibility.
- Automated Client Logo Carousels: Scrolling carousels to showcase client and partner logos, enhancing trust and demonstrating industry presence.
- Interactive Contact & Quote Request Form: A contact form for users to request services or quotes, featuring clear input fields, form validation, and an immediate success feedback message upon submission.
- Content Management Integration: Foundation for managing static content such as services, portfolio items, blog posts, and FAQs via a database backend.

## Style Guidelines:

- Primary Color: Muted brown-orange (#D0A38D). This warm, earthy tone symbolizes reliability, global connection, and a grounded approach to international trade. It's suitable for accents, call-to-actions, and highlights on both light and dark backgrounds.
- Background Color: Subtly warm off-white (#F9F5F3). This color is derived from the primary hue but heavily desaturated and brightened, providing a clean, inviting canvas that resonates with natural, earthy aesthetics, and is consistently used across light sections.
- Accent Color: Deep muted reddish-brown (#993842). This analogous color provides a sophisticated, darker contrast to the primary. It's intended for drawing attention to critical interactive elements or subtle thematic details, maintaining a coherent yet dynamic palette.
- Body and Headline font: 'Montserrat' (sans-serif). A versatile and modern sans-serif font, as provided in the existing stylesheet. Note: currently only Google Fonts are supported.
- Utilize modern, expressive emoji-style icons (e.g., ✈️, 🚢, 🌐, 🏆). This choice injects a friendly, visually engaging, and universal layer into the design, simplifying complex concepts for a diverse international audience.
- Implement a modular, section-based layout with responsive grid structures (as outlined by the 'Socialee' inspired layout). This ensures content is logically organized and adapts seamlessly across various screen sizes, offering a consistent user experience.
- Subtle, fluid animations including scrolling marquees for client logos, counter-up animations for statistics, and elegant hover effects on interactive elements like cards and buttons. These add polish and engagement without being distracting.
